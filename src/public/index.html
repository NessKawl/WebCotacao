Web Cotação

Um sistema para donos de mercados, lojas ou qualquer pessoa que necessite fazer cotação com diferentes fornecedores para chegar em um melhor resultado de compra, buscando sempre comprar pelo melhor preço.


Gerar modelo

INSTRUÇÕES:

  1. Escolha o arquivo em excell desejado;
  2. Informe o identificador (nome do fornecedor, empresa, etc);
  3. Clique em adicionar outra planilha, e repita os passos acima (faça quantas vezes for necessário);
  4. Após colocar todas as planilhas, faça o download do arquivo consolidado.

Obs: em caso de erro, baixe o nosso modelo e monte as cotações com base no arquivo, os nomes das colunas devem ser mantidos.
